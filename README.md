# CodeForge

**CodeForge** is a platform for automating the creation and provisioning of multi-language software projects. Its main goal is to accelerate project bootstrapping by offering customizable templates, Git repository integration, and automated configuration of infrastructure, pipelines, and deployment environments.

## ✨ Overview

CodeForge allows users with different roles — such as developers, tech leads, DevOps engineers, and product owners — to create on-demand projects from version-controlled templates. The platform provides a user-friendly interface for configuring and provisioning resources from the first commit to final deployment.

## 🚀 Features

- 🎯 **Create projects from Git templates** (GitHub, GitLab, Gitea)
- 🌐 **Support for multiple languages and frameworks**
- 🔧 **Configure dependencies and auxiliary services (e.g., Redis, PostgreSQL, RabbitMQ)**
- ☁️ **Integrate with cloud providers (AWS, GCP, Azure, etc.)**
- 🛠️ **Select CI/CD pipelines based on stack**
- 📦 **Automated deployment to Lambda, EC2, Kubernetes, and more**
- 👥 **User role and access level management**
- 📁 **Provision repositories and environments via web interface**

## 📌 Example Flow

1. The user accesses the web interface.
2. Selects a base template and the target Git repository.
3. Defines the desired tech stack and additional services.
4. Chooses a CI/CD pipeline and deployment target.
5. CodeForge generates and provisions the complete project automatically.

## 🧱 Tech Stack (Planned)

- Backend: Golang
- Frontend: Vue.js / Quasar
- Infrastructure: Kubernetes, Docker, Terraform (eventually)
- Integrations: GitHub API, GitLab API, Gitea API, AWS SDK, etc.
